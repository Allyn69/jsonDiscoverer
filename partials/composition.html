How it works?

The composition discoverer infer a set of matching elements from a set of JSON documents. The graph therefore allows you to easily visualize how your APIs can be composed

To use the composition discoverer follow these steps:

  • Step 1. Provision. You have to provide two or more JSON sources. For each source you have to give a name and provide one or more JSON definitions.
  • Step 2. Composition discovery. For each JSON source we will discovery the schema information and then we all (sub)schemas will be composed in a graph where graph nodes represents schema concepts and attributes. We will discover as well possible links among these concepts.

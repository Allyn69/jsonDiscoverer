How does it work?

The composition discoverer infers a set of matching elements from a set of JSON documents (coming from the same or several sources). It generates that allows you to easily visualize how your APIs can be composed.

To use the composition discoverer follow these steps:

  • Step 1. Provision. You have to provide two or more JSON sources. For each source you have to give a name and provide the input (optional) and one or more JSON definitions.
  • Step 2. Composition discovery. For each JSON source we will discovery the schema information and then we all (sub)schemas will be composed in a graph where graph nodes represents schema concepts and attributes. We will discover as well possible links among these concepts.
  • Step 3. Composition Graph. You can apply the layout to better read the graph.
  • Step 4. Calculate a Path. Paths in the graph can represent API compositions. Given a source and a target nodes, you can calculate a path, which we will represent as a sequence diagram.

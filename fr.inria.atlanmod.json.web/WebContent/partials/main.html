What is this?

The JSON discoverer is a tool that allows you to discover the implicit schema of your JSON. JSON definitions include both metadata (i.e., the schema) and data (i.e., the values conforming to the schema). Our tool analyzes your JSON definitions and gives you a class diagram representing the schema plus an object diagram representing the data.

The tool is inspired in the research paper Discovering Implicit Schemas in JSON Data published in the ICWE'13 conference.

Simple Discoverer

The simple discovery mode analyzes a single JSON definition and provides both the schema and the data.

Advanced Discoverer

The advanced mode analyzes sets of JSON definitions and discover the common schema among them.
